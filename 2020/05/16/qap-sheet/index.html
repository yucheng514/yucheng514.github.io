  • 小程序的核心是一个响应式的数据绑定系统，分为视图层和逻辑层。这两层始终保持同步，只要在逻辑层修改数据，视图层就会相应的更新。

  • 关掉自动刷新，重新进入页面的时候，在不重新编译的情况下，axml 文件改动过的内容能生效（包括页面的 { { } } js 代码），js 文件的不能生效。
